Trabalho de Teste de Software. 

Aplicação de teste unitário em uma agregação (Carro). 

Aplicação de teste de integração; 

Pelo menos, um para cada método e interface.

Carro:
- Motor
- Banco
- Acessório
- Painel
- Pneu
- Vidro

Assertions:
- assertTrue
- assertFalse
- assertNull
- assertNotNull
- assertEquals
- assertNotEquals
- assertArrayEquals
- assertIterableEquals
- assertTimeout
- assertTimeoutPreemptively
- assertThrows
